Questions & Answers

1. What is the difference between a normal div and a grid container?
A normal div places elements one below another. A grid container allows arranging items in rows and columns with better layout control.

2. Why does the layout look the same even after applying Grid?
The layout looks the same because no rows or columns are defined. Only display: grid is applied.

3. Are grid items created automatically?
Yes, all child elements inside a grid container automatically become grid items.

4. Can Grid work without rows or columns?
Yes, Grid works but does not show visible changes until rows or columns are defined.

5. What do you think Grid needs to show visible changes?
Grid needs grid-template-columns or grid-template-rows to clearly show the layout structure.
