Questions & Answers

1. Why do we check browser support?
To make sure the feature works properly in the browser.

2. What happens if Grid is not supported?
Normal CSS is used and the layout still works.

3. Is this needed for all projects?
No, it is needed only when old browsers must be supported.

4. Who uses this in real projects?
Developers use it in big or public websites.

5. What does safe CSS mean?
Safe CSS means styles that do not break in any browser.
