Questions & Answers

1. Where does gap add space?
Gap adds space between grid items, not outside the container.

2. How is gap different from margin?
Gap is controlled by the grid container, but margin is applied to each item.

3. Does gap affect container size?
No, gap does not increase container size, only space between items.

4. Can we use gap only for rows or columns?
Yes, we can use row-gap or column-gap separately.

5. Why is gap preferred in Grid layouts?
Gap is easy to use and keeps spacing clean and consistent.
