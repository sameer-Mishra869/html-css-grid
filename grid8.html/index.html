Questions & Answers

1. What problem does auto-fit solve?
Auto-fit automatically adjusts the number of columns based on screen size.

2. Why is minmax important?
Minmax sets minimum and maximum size so boxes don’t get too small.

3. Do items shrink too much?
No, minmax stops items from shrinking below the minimum width.

4. Is this layout mobile friendly?
Yes, columns change automatically and fit small screens.

5. Why are media queries not needed here?
Grid handles responsiveness by itself using auto-fit and minmax.
