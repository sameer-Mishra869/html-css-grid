Questions & Answers

1. Which layout breaks faster on small screens?
Fixed layout breaks faster because px values do not change.

2. Which layout adjusts automatically?
Flexible layout adjusts automatically because it uses fr units.

3. Why is fixed layout risky?
Fixed layout can overflow or cause horizontal scroll on small screens.

4. Where can fixed columns be useful?
Fixed columns are useful for sidebars, icons, or small fixed sections.

5. Which unit would you choose for a responsive website?
I would choose fr because it adjusts with screen size.
