Questions & Answers

1. Why is 12-column grid popular?
Because 12 can be divided easily into 2, 3, 4, or 6 parts.

2. How does spanning help layout?
Spanning lets boxes take more space based on importance.

3. Where is this system used?
It is used in websites, dashboards, and admin panels.

4. Can we create layouts without this system?
Yes, but the 12-column system makes alignment easier.

5. What happens if column count changes?
The layout structure changes and spans must be adjusted.
