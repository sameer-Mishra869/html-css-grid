CSS Grid – Task 4: Grid Rows

Row 1 (Fixed Height)
This row has fixed height.
Row 2 (Auto Height)
This row has more text. When content increases, this row grows automatically. Grid adjusts height based on content.

Questions & Answers

1. What does auto do in grid rows?
auto adjusts the row height based on content.

2. Which row grows based on content?
The row with auto grows when text increases.

3. Can rows have different heights?
Yes, each row can have its own height.

4. What happens if content is very large?
The auto row increases its height to fit the content.

5. Where are grid rows useful in real layouts?
Grid rows are useful for headers, content sections, and footers.
